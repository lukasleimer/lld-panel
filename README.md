# LLD Panel

Internes Verwaltungsportal für **Lukas Leimer Development (LLD)**.

## Ziel

Das Panel dient als zentrale Verwaltungsplattform für:

- Kundenverwaltung
- Projekte
- Website-Templates
- Hosting
- Domains
- Deployments
- Monitoring
- Dokumentation

## Technologie

- Python 3.14
- Django
- PostgreSQL (später)
- Docker (später)
